# 🔐 URL Safety Checker

A cybersecurity tool that analyzes URLs and detects potential phishing risks.

## Features
- Detects IP-based URLs
- Detects shortened links
- Checks suspicious keywords
- Risk scoring system
- Safety percentage
- UI with modern UX

## Tech Stack
- FastAPI (Backend)
- HTML/CSS/JS (Frontend)

## Run Locally
```bash
pip install -r requirements.txt
uvicorn app:app --reload
